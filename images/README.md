# Изображения для проекта "Стулья"

В этой папке должны находиться локальные копии всех изображений, используемых в проекте.

## Рекомендации по оптимизации изображений:

### 1. Форматы изображений
- Используйте формат WebP для лучшего соотношения качества и размера
- Альтернативно, используйте JPEG для фотографий и PNG для изображений с прозрачностью
- Для простых иконок можно использовать SVG

### 2. Размеры изображений
- Оптимальный размер для фотографий персон: 400x600 пикселей (примерно)
- Максимальный размер: 800x1200 пикселей
- Не используйте изображения больше необходимого размера отображения

### 3. Сжатие
- Используйте инструменты сжатия изображений:
  - TinyPNG (https://tinypng.com/)
  - ImageOptim
  - Squoosh (https://squoosh.app/)
  - Optimizilla

### 4. Именование файлов
- Используйте понятные имена: `{person_name}_{before|after}.{jpg|png|webp}`
- Например: `bukharin_before.jpg`, `bukharin_after.jpg`

### 5. Пример оптимизации с помощью скрипта
```bash
# Установка imagemin-cli (если установлен Node.js)
npm install -g imagemin-cli

# Оптимизация всех изображений в папке
imagemin *.jpg --out-dir=optimized/
```

### 6. Заглушка для отсутствующих изображений
Если изображение отсутствует, используйте SVG-заглушку:

```svg
<svg xmlns="http://www.w3.org/2000/svg" width="400" height="600" viewBox="0 0 400 600">
  <rect width="400" height="600" fill="#e0e0e0"/>
  <text x="50%" y="50%" text-anchor="middle" dominant-baseline="middle" 
        font-family="Arial, sans-serif" font-size="16" fill="#666">
    Изображение недоступно
  </text>
</svg>
```

После загрузки и оптимизации изображений, замените файл `data.js` на `data_local.js` в вашем HTML файле.
