# Стулья - Репрессированные деятели СССР 1930-х годов

Это статическое веб-приложение, представляющее репрессированных деятелей СССР 1930-х годов. Проект был преобразован из Next.js приложения в статический HTML/CSS/JS для размещения на GitHub Pages.

## Особенности

- Интерактивный интерфейс для просмотра биографий репрессированных деятелей
- Категории: политические деятели, учёные, писатели, военные и др.
- Возможность просматривать фотографии до и после репрессий
- Поддержка светлой и тёмной темы
- Адаптивный дизайн для различных устройств

## Использование

Для переключения между светлой и тёмной темой:
- Двойной клик на заголовке "Стулья"
- Или нажмите клавишу 'T'

Для просмотра фотографий до и после репрессий, кликните на фотографию персоны.

## Структура проекта

- `index.html` - основная HTML-страница
- `styles.css` - стили приложения
- `script.js` - основная логика приложения
- `data.js` - данные о репрессированных деятелях
